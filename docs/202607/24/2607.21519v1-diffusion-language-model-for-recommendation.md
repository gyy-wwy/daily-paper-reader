## 摘要
大型语言模型（LLM）赋能的推荐系统已成为生成式推荐的一种有前景的范式，利用其强大的语义推理和生成能力来建模复杂多样的用户偏好。然而，现有方法大多依赖自回归范式，该范式对于推荐并非最优。下一个词元的目标强调序列顺序，而非用户偏好背后项目间的结构化依赖关系。此外，前缀约束生成限制了双向上下文，并固守从左到右的解码方式，导致早期错误累积而无法纠正。受扩散语言模型成功经验的启发，我们提出DLMRec，一种专为推荐量身定制的离散扩散语言模型，为自回归生成提供了一种有吸引力的替代方案。具体而言，DLMRec引入了三个关键组件以桥接扩散语言建模与推荐。首先，一种协同感知的随机分词器将多跳协同信号编码为与扩散建模兼容的表达性离散词元。其次，一种课程驱动的训练策略通过渐进的项目级和词元级学习使去噪过程与偏好恢复对齐。第三，一种稳定性感知的投票机制聚合迭代预测，以提高生成的连贯性和鲁棒性。

## Abstract
Large language model (LLM)-empowered recommender systems have emerged as a promising paradigm for generative recommendation, leveraging their strong semantic reasoning and generative capacity to model complex, diverse user preferences. However, most existing approaches rely on an autoregressive paradigm that is suboptimal for recommendation. The next-token objective emphasizes sequential order rather than the structural inter-item dependencies underlying user preferences. In addition, prefix-constrained generation restricts bidirectional context and commits to left-to-right decoding, causing early errors to accumulate without correction. Inspired by the success of diffusion language models, we propose \textbf{DLMRec}, a discrete diffusion language model tailored for recommendation that offers a compelling alternative to autoregressive generation. Specifically, DLMRec introduces three key components to bridge diffusion language modeling with recommendation. First, a collaborative-aware stochastic tokenizer encodes multi-hop collaborative signals into expressive discrete tokens compatible with diffusion modeling. Second, a curriculum-driven training strategy aligns the denoising process with preference recovery through progressive item- and token-level learning. Third, a stability-aware voting mechanism aggregates iterative predictions to improve generation consistency and robustness.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有基于大型语言模型（LLM）的推荐系统大多采用**自回归生成范式**（如 GPT 系列），该范式通过“下一个词元预测”目标来建模用户行为序列。然而，推荐任务中用户偏好的本质是**物品间的结构化依赖关系**，而非交互顺序。自回归的严格左到右生成导致两个主要缺陷：
  1. **目标不匹配**：下一词元目标强调局部顺序，忽略了用户偏好中物品间的全局组合关系。
  2. **误差累积**：前缀约束生成使早期预测错误无法纠正，且无法利用未来双向上下文。
- **整体含义**：论文提出**扩散语言模型（Diffusion Language Model）** 作为自回归的替代方案。扩散模型通过双向去噪和迭代精炼，天然适合捕获推荐场景中的结构依赖和全局偏好，并能从稀疏、有噪声的观测中恢复潜在偏好。
- **初步实验验证**：在 MovieLens-1M 上对比离散扩散模型与自回归模型，扩散模型在 Recall 和 NDCG 上表现更优，训练更稳定。

### 关键技术细节

#### （1）协同感知随机分词器（Collaborative-Aware Stochastic Tokenizer, CAST）
- **目的**：将用户-物品交互图的**多跳协同信号**转化为离散 token，且兼容扩散模型的双向并行处理。
- **过程**：
  - 使用 LightGCN 编码用户/物品在不同传播深度（hop）的表示（0-hop 到 K-1-hop）。
  - 为每个 hop 维护一个独立的子码本（sub-codebook）。
  - **随机量化**：计算 hop 表示与码本中 top-S 个最相似码字的余弦相似度，再经 softmax 生成分配概率（温度 τ 随 hop 增加而增大，低 hop 尖锐、高 hop 模糊），然后采样得到离散 token 索引。
  - 将多个 hop 的 token 拼接，通过解码器重构原始嵌入。
- **训练目标**：重建损失 + 码本损失 + 承诺损失 + 对比学习损失（拉近重构嵌入与原始嵌入，推远其他样本）。

#### （2）课程驱动的语义对齐训练策略（Curriculum-Driven Semantic Alignment）
- **目的**：使扩散语言模型逐步学会从协同 token 和文本描述中恢复偏好。
- **两阶段课程**：
  - **阶段一：物品级掩码对齐**。将历史物品的 token 块和文本描述块**整体掩码**（而不是独立掩码 token）。采用从易到难的课程：早期轻掩码，后期重掩码，防止模型学到无用的词表映射。目标：建立 token 与文本语义的对应关系。
  - **阶段二：词元级掩码与偏好感知去噪**。对目标物品的所有 token 独立线性掩码（从 t=0 到 T）。除标准扩散重建损失外，引入**偏好损失**：通过软量化（soft quantization）将目标位置 logits 转换为连续表示，然后计算与正样本（真实物品）和硬负样本（top-5 相似非交互物品）的对比排名损失。
- **训练循环**：交替执行阶段一和阶段二多次。

#### （3）稳定性感知投票机制（Stability-Aware Voting）
- **目的**：在推理时，聚合多次迭代精炼的中间预测，提升最终推荐的一致性。
- **流程**：
  - 初始预测后，进行 S 次迭代精炼。对每个目标 token 位置 k：
    - 若当前预测与前一步相同且置信度 > 阈值 γ，则标记为稳定，直接用 argmax token 的嵌入作为下一步输入。
    - 否则为不稳定，将当前预测嵌入与用户引导先验（mask 嵌入 + 用户 token 嵌入）按置信度混合，作为下一步输入。
  - 最终：稳定位置直接取 argmax token；不稳定位置对全轨迹的 log-probability 求和投票。
  - 将最终 token 序列解码为物品表示，在物品库中检索最近邻作为 Top-N 推荐。

### 公式/算法流程（文字说明）
- 扩散前向过程：以吸收态（mask token）方式逐步替换 token。
- 训练目标：掩码位置上的交叉熵损失（标准扩散） + 第二阶段新增的偏好对比损失。
- 推理：逐步精炼 + 稳定性投票。

## 6. 论文的主要结论与发现

- DLMRec 在三个数据集上**一致超越所有基线**，相比最强 LLM 基线（LLaRa）平均 Recall 提升 6.75%，NDCG 提升 5.19%；相比离散扩散基线（LLaDaRec）提升更大。
- **扩散语言模型是自回归生成的有力替代**，尤其适用于推荐中的结构依赖建模。
- 各组件均有效：CAST 优于标准 VQ-VAE；课程训练（物品级+词元级）不可或缺；偏好损失提升判别能力；稳定性投票机制稳定生成。
- 推理时仅需少量迭代（2-3步）即达最优，效率可与自回归相当或更优（同 token 数量下）。
